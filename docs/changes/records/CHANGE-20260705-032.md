## 任务名称

个股详情页结构状态因子面板升级至 V1.8（补齐 dsa_segment/swing/cost/volatility/participation/relation 约 50 字段）

## 需求出处

- 用户 `/plan` 指令：在 PR #19 已合并的 V1.7 结构状态面板基础上，补齐"结构状态因子层 V1.8"，覆盖 5 组因子 + relation 区块约 50 个新字段。
- 关键约束：继续使用 `GET /api/v1/instruments/{id}/structural-factors`；仍按需计算单只股票 1d + 15m 最新已完成 bar；不新增全市场/全历史预计算、不新增重型 worker、不新增大表；只使用 volume，禁止 amount；前端只渲染 DTO；15m 只用 completed bars；swing 只用 confirmed pivot（无未来函数）；DSA/SQZMOM/BB/Node 复用现有 SSOT。
- 设计文档：`.trae/documents/structural-factors-v1-8-complete.md`（用户确认）。

## 修改前行为

- V1.7 仅 5 组基础字段 + `relation.momentum_alignment` 事件字段。
- `dsa_segment.segment_extents_pct` 用 `dsa_vwap` 当 `last_close` 计算段收益，不够准确。
- 缺少段级收益/斜率/效率/段间对比、swing range/position/retracement、cost price_vs_poc/value_area_position/nearest_node_*/distance_to_node_*_atr/node_*_strength、volatility distance_to_bb_*_atr/sqz_on/sqz_off/sqzmom_abs_percentile、participation 段级成交量、relation 客观关系字段。

## 修改后行为

### 后端 `structural_factor_service.py`

- **dsa_segment**（V1.8 扩展）：
  - 基础：`dsa_value`、`price_vs_dsa_atr`；
  - 当前段：`current_dsa_segment_id`、`current_dsa_segment_dir`、`current_dsa_segment_age_bars`、`current_dsa_segment_return_pct`、`current_dsa_segment_slope_pct_per_bar`、`current_dsa_segment_slope_atr_per_bar`、`current_dsa_segment_efficiency_0_1`、`current_segment_volume_sum`；
  - 前一段：`prev_dsa_segment_dir`、`prev_dsa_segment_age_bars`、`prev_dsa_segment_return_pct`、`prev_dsa_segment_slope_pct_per_bar`、`prev_dsa_segment_slope_atr_per_bar`、`prev_dsa_segment_efficiency_0_1`、`prev_segment_volume_sum`；
  - 段间对比：`segment_return_abs_ratio`、`segment_slope_abs_ratio`、`segment_duration_ratio`、`segment_efficiency_delta`、`current_vs_prev_volume_ratio`、`current_segment_return_per_volume`、`prev_segment_return_per_volume`、`return_per_volume_ratio`、`volume_per_1pct_return`；
  - 段收益/斜率/效率一律基于 `close` 或段实际端点价格，不再用 `dsa_vwap` 替代（修复 V1.7 bug）。
- **swing_position**（V1.8 扩展）：`swing_range`、`price_position_in_swing_0_1`、`distance_to_swing_high_atr`、`distance_to_swing_low_atr`、`retracement_from_high_0_1`、`rebound_from_low_0_1`、`bars_since_swing_high`、`bars_since_swing_low`。
- **cost_position**（V1.8 扩展）：`price_vs_poc_atr`、`value_area_position_0_1`、`nearest_node_above_price`、`nearest_node_below_price`、`distance_to_node_above_atr`、`distance_to_node_below_atr`、`node_above_strength`、`node_below_strength`（node_strength 从 `peak_df` 按 `price_mid` 查找）。
- **volatility_momentum**（V1.8 扩展）：`distance_to_bb_upper_atr`、`distance_to_bb_lower_atr`、`sqzmom_abs_percentile`、`sqz_on`、`sqz_off`；保留 `bb_percent_b`/`bb_bandwidth_pct`/`bb_bandwidth_percentile`/`sqzmom_val`/`sqzmom_delta_1`/`sqzmom_percentile`。
- **participation**（V1.8 扩展）：保留 `volume_ratio_20`/`volume_percentile_120`；新增段级成交量字段（从 `dsa_segment` 共享）：`current_segment_volume_sum`/`prev_segment_volume_sum`/`current_vs_prev_volume_ratio`/`current_segment_return_per_volume`/`prev_segment_return_per_volume`/`return_per_volume_ratio`。
- **relation**（V1.8 重写）：移除 `momentum_alignment` 事件字段，改为客观关系字段 `primary_dir`/`secondary_dir`/`trend_alignment`/`primary_swing_position`/`secondary_swing_position`/`primary_slope_atr`/`secondary_slope_atr`/`secondary_vs_primary_position_delta`。
