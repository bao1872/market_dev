- release 候选必须从最新 `origin/main` 创建，使用 `--no-ff` merge 合并修复分支，形成具有两个 parent 的 merge commit；

- `CHANGELOG.md` 索引必须根据真实 git 历史填写 Base/Head/Merge Commit。
