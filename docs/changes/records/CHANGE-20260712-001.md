## 修改后行为
1. 新增 `backend/app/research/regime_discovery/` 模块（8 文件：`data_access` / `feature_builder` / `distribution_audit` / `preprocessing` / `models` / `stability` / `reporting` / `__init__`）；
2. 新增 CLI `backend/scripts/research_regime_discovery.py`；
3. 新增测试 `backend/tests/research/`（8 测试文件 + conftest，96 用例）；
4. 只读 `research_feature_matrix_rows` + `bars_daily`，派生 17 个聚类特征，双表示（absolute robust + cross-sectional rank）；
5. 主模型 MiniBatchKMeans k=3..8（n_init=10, max_iter=100）+ 辅助 diagonal-covariance GMM；
6. 稳定性门槛（silhouette >= 0.08 / bootstrap ARI >= 0.60 / centroid cosine >= 0.85 / 最小簇占比 >= 3% / 最大簇占比 <= 60%）；
7. 输出 manifest + 7 CSV + report.md，≤ 50MB，保留最近 3 次；
8. 资源约束：float32、chunk 25,000、单进程、OMP/OPENBLAS/MKL threads=1、RSS ≤ 1.5GB；
9. 事务 `READ ONLY` + `statement_timeout=120s`，不修改生产。
