## 2. 权限核对要点

- 核心业务 API 必须 active subscription；
- Admin API 必须 admin；
- Capture API 必须 Capture Token；
- 消息/渠道必须按 JWT user_id 所有权隔离；
- 到期用户只允许历史消息只读和续期相关能力。
