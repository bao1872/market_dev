# CHANGE-20260709-012

## 任务名称
Swing 当前状态语义修正 + Temporal V1 位置污染修复 + capture/飞书右侧空白修复

## 需求出处
当前线上 Swing 摘要卡把 confirmed pivot 当作当前状态，强趋势段 `price_position_in_swing_0_1` 会突破 `[0,1]`；Temporal V1 `m15_position_relative_to_daily` 被 confirmed raw 污染；`capture=feishu` 截图模式右侧仍有空白侧栏。

## 修改前行为
1. `StockStructuralStatePanel` Swing 摘要卡显示 `developing_swing_*` 字段；
2. `temporal_feature_service._compute_derived_relation` 使用 `daily/m15_price_position_in_developing_swing_0_1`；
3. `capture=feishu` 布局未强制隐藏 `.tv-side-column`，右侧存在空白。

## 修改后行为
1. Swing 摘要卡改用 V1.9 active swing 字段（`active_swing_dir`/`active_swing_high`/`active_swing_low`/`price_position_in_active_swing_0_1` 等），confirmed pivot 只保留在明细 JSON；
2. `m15_position_relative_to_daily` 使用 `daily/m15_price_position_in_active_swing_0_1`（clip `[0,1]`），不回退 confirmed raw；
3. `capture=feishu` / `capture=1` / `hideStructuralState=1` 触发 `isCaptureMode=true`，`.tv-side-column` 隐藏，`.tv-chart-column` 占满宽度。
