## Node Cluster / Volume Node 参数

| 参数名 | 值 | 说明 |
|--------|-----|------|
| NODE_CLUSTER_PRIMARY_PERIOD | 1d | 主周期 |
| NODE_CLUSTER_PRIMARY_BARS | 250 | 主周期根数（最近 N 根已完成前复权日线） |
| NODE_CLUSTER_LOW_PERIOD | 15m | 低周期 |
| NODE_CLUSTER_LOW_BARS | 1200 | 低周期根数 |
| NODE_CLUSTER_MINUTE_BARS | 2 | 分钟线根数（穿越检测） |
| VP_ROWS | 100 | Volume Profile 行数 |
| VP_VALUE_AREA_PCT | 0.7 | 价值区域占比 |
| VP_PEAK_DETECTION_PCT | 0.05 | 峰值检测阈值 |
| VP_NODE_THRESHOLD_PCT | 0.01 | 成交量节点阈值 |
| VP_TROUGHS_SHOW | none | 波谷显示模式 |
| VP_TROUGHS_DETECTION_PCT | 0.07 | 波谷检测阈值 |
| VP_HIGHEST_N_NODES | 0 | 最高 N 个节点（0=不限制） |
| VP_LOWEST_N_NODES | 0 | 最低 N 个节点（0=不限制） |
| NODE_CLUSTER_EVENT_TTL_SECONDS | 600 | 事件 TTL（秒） |

## 刷新时点

- 日线 / 15m / 60m：每个交易日 16:00 定时刷新（CronTrigger hour=16 minute=0, Asia/Shanghai）
- 周线 / 月线：不存储 DB，从日线动态合成（convert_kline_frequency），不参与定时刷新
- 1m：不在定时调度范围（调度仅覆盖 d/15m/60m），由监控策略实时拉取
