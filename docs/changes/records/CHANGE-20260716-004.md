# CHANGE-20260716-004: AFC V1 原子值 UI 改造（短原子值 + visualKind 统一枚举 + 持久化/调试分离 + 前端状态观察重构）

## 变更编号
CHANGE-20260716-004

## 任务名称
Atomic Fact Contract V1 原子值 UI 改造：把后端「长句 valueText」+ 前端「卡片嵌套行」改为结构化短原子值状态观察 UI。

## 需求出处
`ref/PROMPT.md`（AFC V1 原子值 UI 改造需求）+ 项目负责人确认。

## 修改前行为
- 后端 `valueText` 为完整中文长句（如「主趋势方向为上行」「尚未到达 2.5000 ATR」），精度散落 `.4f/.6f/.2f`。
- `visualKind` 枚举为 `value/category/relation/position/distance/ratio`，前端需解析中文推断状态。
- M5 文案为「挤压中/释放中」；距离类方向靠 `valueText.includes('已越过')` 解析。
- `compute_atomic_facts` 单一函数返回含 `debug` 数组；`feature_snapshot_service` 直接持久化全量（含 debug）。
- `_is_valid_stored_afc` 仅校验 publicKey 存在，不校验版本字段。
- `recentChanges` 缺中文 `label`，前端显示 `publicKey`（内部键泄露）。
- 前端 `.factRow` 为嵌套小卡片（背景/边框/圆角）；S3 轨道仅 0.33/0.67 刻度无低位/高位/数值标注；Auxiliary 平铺无分组；Drawer 无焦点管理。

## 修改后行为
### 后端展示契约
- `valueText` 改为短原子值：T1=`上行/下行/中性`，T2=`+0.0123`（+secondaryText `ATR / 根日K`），T4=`18`（+`个交易日`），T5=`1.23×`（+`分类未启用`），M1/M5/S1/S2=仅 categoryLabel，M2=`+0.002`，M3=`+0.000300`（+categoryLabel），S3=`0.63`（轨道+`0.63 · 中间`），S7/S8=`1.23 ATR`（+categoryLabel `尚未到达/已越过`），V3=`1.11×`（+`分类未启用`）。
- 统一格式器 `_fmt_atomic_value(fact_id, value)` 读 presentation `valuePrecision`，ratio→`1.23×`，distance→`1.23 ATR`（绝对值），signed（T2/M2/M3）→`+` 前缀，禁止散落 `.4f/.6f`。
- `visualKind` 统一枚举：`metric | value_with_category | relation | position | distance | ratio`（旧 `value`→`metric`，`category`→`value_with_category`）。前端禁止解析中文推断类型/状态。
- presentation JSON：T5/S3/S7/S8/V3 `valuePrecision` 4→2（`1.23×`/`0.63`/`1.23 ATR`）。
- M5 修复：`sqz_on`/`sqz_off` 任一缺失即缺失；双 true→缺失 + `m5_inconsistent` warning；categoryLabel `ON→正在收紧 / OFF→正在释放 / NORMAL→正常`。
- `recentChanges` 每条含中文 `label`（来自 presentation `publicLabel`）；relation 类 `fromText/toText` 回退 categoryLabel；前端不再显示 `publicKey`。

### 持久化与管理员调试分离
- `compute_atomic_facts()` 仅返回 `core/auxiliary/availability`（**不含 debug**）。
- `compute_atomic_fact_debug()` 仅返回 debug 列表（管理员按需即时生成）。
- `build_persisted_afc_payload()` 包装 `payloadVersion=1`/`researchContractVersion`/`researchFreezeVersion=V4.13`/`presentationVersion` + `core/auxiliary/availability`，不含 debug。
- `feature_snapshot_service.build_summary_payload` 改用 `build_persisted_afc_payload`（持久化 payload 含四版本字段，无 debug）。
- `stock_context._is_valid_stored_afc` 严格校验：四版本字段（与当前常量一致）+ 四组结构（core dict/auxiliary list/availability coreDenominator=14）+ publicKey 存在 + **不含 debug**。任一不满足→fallback 重算（不回写旧快照）。
- 管理员 debug 走 `compute_atomic_fact_debug(snapshot.structural_payload, snapshot.temporal_payload)` 即时生成（admin 仍返回完整 factId/sourcePath/rawValue/thresholdRef/featureFlag/missing）。
- persisted-first 与 fallback 公开结果一致（均新格式）；GET 零写入；V1/T3/T6 规则不变；旧 worker 镜像写入的旧格式（缺版本/含 debug）由 validator 判定 fallback，保证 API 兼容。

### 前端重构
- `endpoints.ts`：`AtomicFactItem.visualKind` 新枚举；`valueText: string | null`；`AtomicFactChange.label: string`。
- `AtomicFactsPanel.tsx`：`FactRow` 按 `visualKind` 渲染（metric 数值 mono+高字重+secondaryText；value_with_category 数值+categoryLabel 徽章；relation 仅 categoryLabel 一次；distance 徽章+数值各一次；ratio 数值+secondaryText）；S3/S6 完整轨道（低位/0.33/0.67/高位 + 圆点 + `0.63 · 中间`）；Auxiliary 按 动量补充/结构补充/成交补充 分组默认收起；RecentChanges 显示中文 label（非 publicKey）。
- `AtomicFactsPanel.module.scss`：`.factRow` 改 CSS Grid `minmax(0,1fr) auto` + `grid-template-areas "label value" / "secondary secondary"`，背景透明仅底部分隔线（非嵌套卡片）；S3 轨道完整刻度；Auxiliary 分组样式。
- `AtomicFactsDrawer.tsx`：打开聚焦关闭按钮、焦点 trap（Tab/Shift+Tab 限制在抽屉内）、关闭恢复打开前焦点、body 滚动锁定、Escape/遮罩/关闭按钮关闭。
