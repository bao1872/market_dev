- `/market` 渲染 `MarketWorkspacePage`（三栏布局）：

  - 右栏 `StockStructuralStatePanel`：可收起；收起时不挂载、不请求 structural/temporal 数据。
- URL 状态：`/market?scope=watchlist|market&symbol=xxx&timeframe=1d`；scope/symbol/timeframe 进 URL；右栏折叠和 viewport 留本地。
- 切换股票不整页刷新（改 URL symbol 参数，React Query 缓存复用）。
- `useStockResearchData` 集中 bars/indicators/quote/events/watchlist/memo 请求与组装。
- `detailNavigation.resolveBackPath` watchlist fallback 改为 `/market?scope=watchlist`。
- 图表 timeframe 仅展示，不改变 1d+15m 监控配置或 1m 事件触发口径。
- 不新增后端 API；CaptureStockPage 独立不变；不删除 IndexPage/ScreenerPage/CaptureStockPage。
