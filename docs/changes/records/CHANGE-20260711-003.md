# CHANGE-20260711-003: StockDetailPage 共享研究核心重构（阶段四）

## 变更编号
CHANGE-20260711-003

## 任务名称
StockDetailPage 降为路由适配器，/market 和 /stock/:symbol 共享研究数据与 K 线核心

## 需求出处
advice.md 阶段四：让 /market 和 /stock/:symbol 真正共用 useStockResearchData + StockResearchWorkspace，保留详情页全部现有功能。

## 修改前行为
- `StockDetailPage`（813 行）独立调用 useBars/useIndicators/useRealtimeQuote/useInstrumentEvents/useWatchlist/useBatchInstruments/useStockMemo 等 hooks 自行组装数据
- 与 /market 的 `useStockResearchData` + `StockResearchWorkspace` 形成两条重复请求链路
- `useStockResearchData` 反向从 `marketWorkspaceUrlState` 导入 `DisplayTimeframe` 类型（stock-research → market-workspace 依赖方向不合理）
- StockDetailPage 的 `quoteStatus` 在 `source === 'daily_fallback'` 时显示"日线回退"（违反 AGENTS §12.14 规则 14）
- StockDetailPage 的 `barsStatus` 使用简化文案，partial 不含当前周期

## 修改后行为
- `StockDetailPage`（453 行）降为路由适配器：解析 URL → 调用共享 hooks → 渲染 header + StockResearchWorkspace + 结构面板 + modals
- 新建 `stockResearchTypes.ts`：DisplayTimeframe/ResearchSource/ALLOWED_TIMEFRAMES/BARS_COUNT_BY_TIMEFRAME/defaultStrategyForSource/normalizeDisplayTimeframe/normalizeResearchSource 权威定义
- `marketWorkspaceUrlState.ts` 从 `stockResearchTypes` 导入并重新导出共享类型，依赖方向修正为 market-workspace → stock-research
- `useStockResearchData` 新增 `priceSummary`/`quoteStatus`/`barsStatus`/`isRenderReady` 返回值，统一 /market 和 /stock 的行情状态文案
- `StockResearchWorkspace` 新增 `toolbar`/`rightPanel`/`showRightPanel`/`chartColumnProps` 可选 props，支持详情页结构面板开关和截图模式属性
- 新建 `useStockDetailActions.ts`：自选列表/加入移出自选/上下切换/memo 读取保存删除（详情页专属）
- 新建 `useStockDetailFeishu.ts`：飞书截图任务创建/轮询/超时/清理（详情页专属）
- `StockDetailPage` timeframe 改为 URL 控制（与 MarketWorkspacePage 一致）
- `quoteStatus` 统一：daily_fallback 显示"行情回退"（不显示"日线回退"）
- `barsStatus` 统一：partial 文案含当前周期（如"K线含未完成 bar（15m）"）
