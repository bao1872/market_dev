# CHANGE-20260708-054

## 变更编号
CHANGE-20260708-054

## 任务名称
research feature matrix causality registry

## 需求出处
用户要求新开 PR：把因果口径定义清楚，并建立研究矩阵骨架。生产 snapshot 与研究矩阵分开，字段命名带 namespace（causal / confirmed_delay / hindsight / label），禁止把 hindsight 或 label 字段当成 causal feature。不新增大表，不跑历史回补，不删除数据库卷或运行中镜像。

## 修改前行为
- 系统只有生产 `stock_feature_snapshots` 矩阵，没有研究矩阵；
- 因果口径（causal / hindsight / label）没有显式登记，存在未来信息泄漏到回测 feature 的风险；
- DSA 字段没有双轨区分（当时可知 vs 未来确认后回标注）；
- Node Cluster 字段没有 hindsight 命名空间标注；
- 历史 full snapshot 回补仍 BLOCKED（PR #41 126min > 120min 阈值），缺少研究侧替代路径。

## 修改后行为
1. 新增 `backend/app/research/feature_causality_registry.py`：
   - `FeatureSpec` frozen dataclass：`key` / `namespace` / `source` / `allowed_for_backtest` / `compute_policy` / `notes`；
   - `__post_init__` 校验 `namespace`/`source`/`compute_policy` 非空 + `key` 必须以 `{namespace}.` 开头；
   - `FeatureCausalityRegistry`：`register`（重复 key 抛 `ValueError`）/ `get` / `all` / `by_namespace` / `keys`；
   - `build_default_registry()` 登记共 27 个字段：
     - causal 10：atr/bb/sqzmom/volume_ratio_20/volume_percentile_120/active_swing/developing_swing/dsa_confirmed_segment/dsa_confirmed_direction/dsa_confirmed_age_bars（policy=series_once, backtest=True）
     - confirmed_delay 4：confirmed_swing_high/confirmed_swing_low/bars_since_confirmed_swing_high/bars_since_confirmed_swing_low（policy=confirmed_only, backtest=True）
     - hindsight 6：dsa_finalized_segment/dsa_finalized_direction/dsa_finalized_age_bars/node_cluster_label/node_cluster_support/node_cluster_resistance（policy=hindsight_once, backtest=False）
     - label 7：future_return_5d/future_return_10d/future_return_20d/future_max_drawdown_10d/future_max_drawdown_20d/breakout_success_10d/failure_breakdown_10d（policy=future_label, backtest=False）

2. 新增 `backend/scripts/research_feature_matrix_backfill.py`：
   - CLI 参数：`--start`（必填）/ `--end`（默认 `latest`）/ `--symbols` / `--limit-instruments` / `--dry-run` / `--output` / `--include-hindsight`（默认 `true`）/ `--include-labels`（默认 `true`）；
   - `_resolve_scope(symbols, limit_instruments)`：任一过滤启用 → `sample`，都未启用 → `full`；
   - `build_plan(args)` 基于 `build_default_registry()` 统计各命名空间字段数，根据开关调整 hindsight/label 计数；
   - `_validate_output_scope(args)`：`--output` 必须配合 sample scope，否则抛 `ValueError`（禁止无过滤全市场输出文件），dry-run 也要校验；
   - `main()` 骨架行为：
     - dry-run：打印计划 + 字段分类统计，不写 DB，不写文件；
     - 非 dry-run 无 `--output`：只打印计划（骨架阶段不实际计算）；
     - 非 dry-run 有 `--output`：校验 sample scope 通过后，骨架阶段不实际写文件；
   - 不接入 `watchlist_ready`，不修改 production snapshot，不新增数据库表。

3. DSA 双轨口径：
   - `causal.dsa_confirmed_segment`/`direction`/`age_bars`：当时已确认状态，可用于回测；
   - `hindsight.dsa_finalized_segment`/`direction`/`age_bars`：未来确认后回标注，禁止进入回测；
   - registry 必须同时登记两类，缺一视为口径不完整。

4. Node Cluster 后验口径：
   - 只能是 `hindsight.node_cluster_label`/`support`/`resistance`；
   - 允许未来信息，只做结构标注；
   - 不得进入 causal，不得接入 watchlist 或生产筛选，不得作为回测 feature。

5. confirmed_swing 口径：
   - `confirmed_delay.confirmed_swing_high`/`low`/`bars_since_*`：只能在确认 bar 生效，不回填 anchor date；
   - 不得作为 hindsight 默认回填；
   - 允许回测，但必须遵守 confirmed_only 计算策略。

6. 新增 `docs/current/06-research-feature-matrix.md`：因果口径表 + 4 namespace 定义 + DSA 双轨 + Node Cluster 口径 + label 不得作为 feature + production snapshot vs research matrix 区别。

7. 更新 `docs/current/MANIFEST.md`：单一事实源表新增 `06-research-feature-matrix.md` 行。

8. 更新 `docs/current/03-jobs-integrations-operations.md`：新增 2.4.2 节「研究特征矩阵脚本骨架」，明确历史 full snapshot 回补仍 BLOCKED，研究回补改走 research matrix。

9. 更新 `docs/current/05-testing-acceptance.md`：新增 3.8 节「研究特征矩阵因果口径回归」（blocking）。

## 风险
- registry 当前是静态登记，后续若需要从代码动态扫描字段需要扩展（本 PR 不在范围）；
- 研究矩阵脚本骨架未实现实际计算，后续需要按 5 步顺序实现（causal rolling → confirmed_delay swing → DSA 双轨 → Node Cluster → labels）；
- `--include-hindsight`/`--include-labels` 当前只影响统计计数，后续实现计算后才会真正过滤字段。

## 遗留问题
- 下一步进入 rolling causal features 实现（ATR/BB/SQZMOM/volume）；
- DSA 双轨计算逻辑需要在 `dsa_selector` 服务侧实现 `causal.dsa_confirmed_*` 与 `hindsight.dsa_finalized_*` 的双轨输出；
- Node Cluster hindsight 标注需要在 `volume_node_monitor` 侧实现；
- label 生成需要 `research_label_service` 实现（当前 source 标为占位，未实现）；
- 历史 full snapshot 回补仍 BLOCKED（PR #41 126min > 120min 阈值），与 research matrix 是独立路径。
