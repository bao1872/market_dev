## 修改前行为

1. 管理员入口：`getAccountMenuItemsForVariant(isAdmin, 'user')` 过滤了 `/admin`，管理员在行情页看不到"管理后台"入口，只能手输 URL。
2. 行业/概念 preset：`applyPresetConfig` 不恢复 `industry`/`concept`；`TablePresetMenu` 不保存板块字段；preset 失效字段静默丢失。
3. 数量契约：部分文档描述"source_total 与筛选使用相同条件"，与实际实现不符。
4. BrandLogo 使用自行绘制的 SVG 折线，未使用批准 Logo 资产。
5. 视觉残留：LandingPage/BetaApplicationModal 主按钮、global.scss focus/selection/toggle 仍使用旧蓝色。
6. AdminRoute 时序：刷新 `/admin` 时 auth store hydration 未完成可能提前判定 false。

## 修改后行为

### 1. 管理员后台入口修复（P0）

- `getAccountMenuItemsForVariant(isAdmin, 'user')`：is_admin=true 额外显示"管理后台"链接 `/admin`；is_admin=false 只显示消息+设置。
- `getAccountMenuItemsForVariant(isAdmin, 'admin')`：显示消息+设置+返回行情；不重复"管理后台"。
- 权限只用 `user.is_admin`，禁止邮箱/role/前端开关推断。
- 普通用户 DOM 不渲染管理后台入口。
- `AdminRoute` 增加 `accessLoading`：auth hydration 或 `/me/access` 未返回时显示 loading，不提前判定 false。
- AdminAppShell 左侧导航保持六项。

### 2. 行业/概念 preset 集成

- `StrategyDataTable.applyPresetConfig` 恢复 industry/concept 到外部受控 state。
- `TablePresetMenu` configToPayload/payloadToConfig 增加 industry/concept 持久化。
- `MarketWorkspacePage` 构建 boardsValidation 传入 StrategyDataTable。
- preset 校验：available=true 且值不在目录 → 跳过应用并 toast 一次；available=false → 保留值但禁用；无 boardsValidation → 直接应用。

### 3. 数量契约修正

- source_total：published run 原始总量，不受业务筛选影响。
- universe_total：all/watchlist 范围总量，业务筛选前。
- filtered_total：keyword+industry+concept+metric_filters 后总量。
- items：filtered_total 当前页。

### 4. 批准 Logo

- BrandLogo 改用批准 PNG 资产（logo_symbol_128.png / logo_horizontal_dark.png）。
- 停用手绘 SVG。

### 5. 视觉 V1.0 残留清理

- LandingPage/BetaApplicationModal 主按钮改为品牌绿渐变。
- global.scss focus/selection/toggle 改为 var(--brand)。
