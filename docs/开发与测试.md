### 9.1 xfail 管理规则

- 任何 `@pytest.mark.xfail` 必须在 `backend/tests/allowlist.json` 中登记
- `allowlist.json` 每条记录必须包含 `test`、`reason`、`issue`（真实 GitHub issue URL 或编号）、`owner`（当前固定 `bao1872`）、`expires`（未来 ISO 日期）
- `backend/pyproject.toml` 已设置 `xfail_strict = true`，xpass 会自动失败
- 超过 `expires` 日期的允许项也会触发 CI 失败
