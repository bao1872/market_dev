1. 状态：**已完成**（不阻塞 UI 修复）

探测环境：
- pywencai 0.13.1（仅安装在 /tmp/pywencai_probe/pylib，**未加入生产依赖**）
- 探测脚本：/tmp/pywencai_probe/probe.py + /tmp/pywencai_probe/probe_diag.py
- 探测结果：/tmp/pywencai_probe/probe_result.json
- 不连接 backend 数据库（不写生产库）
- 不调用 board_sync_service（不开启同步）

探测结论：
- pywencai 0.13.1 import 成功
- 5 项查询（market_info/industry_boards/concept_boards/industry_members）全部失败
- 直接 HTTP 请求 `http://www.iwencai.com/customized/chart/get-robot-data` 返回 **HTTP 401**
- 响应 body：`{"code":0,"msg":null,"data":{"captcha_url":"http://www.iwencai.com/ac_verification/captcha/?host=..."}}`
- iwencai.com 要求验证码验证，pywencai 无 cookie/captcha 绕过能力
- pywencai `convert()` 在 `params.get('data')` 处崩溃（params 为 None），错误处理不完善
- 与 THS 反爬拦截（ALIGN-041）同源：iwencai.com 也拦截物理机出口 IP

结论：pywencai 在当前物理机环境下**不可用**作为板块目录+成分股 provider。
- 与 qstock/THS 同样被反爬拦截
- 需要有效 cookie + captcha 验证才能使用
- 不建议加入生产依赖（即使解决 captcha 也是不稳定方案）
- 维持 BOARD_SYNC_ENABLED=false 现状
- ALIGN-041 OPEN 持续
