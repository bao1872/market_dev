# DSA 选股计算性能基准测试报告

| 项目 | 值 |
|---|---|
| 测试日期 | 2026-07-02 |
| 测试脚本 | `backend/tools/dsa_benchmark.py` |
| DSA 版本 | 1.4.1（released） |
| 样本规模 | 350 只代表性活跃 A 股股票 |
| 运行环境 | APP_ENV=production，只读运行 |
| 数据源 | 生产环境 Docker PostgreSQL 容器（运行中） |

> 注：本次基准测试在生产容器上以只读方式执行，未修改任何生产数据。报告中不包含任何数据库连接凭据或 URL。

## 4. 结论

- 350 只代表性活跃 A 股股票全部完成冷/热启动计算，无失败、无跳过。
- 冷热启动耗时分布接近，p50 均在 130ms 左右，p99 均在 225–232ms 之间。
- 单次完整基准测试总耗时约 105.78 秒，平均每股约 302ms（含行情加载与两次 DSA 执行）。
- DSA 计算阶段本身满足生产环境对单股延迟的预算要求。
