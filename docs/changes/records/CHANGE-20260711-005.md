# CHANGE-20260711-005: 统一行情工作区 P0/P1 收口修正

## 变更编号
CHANGE-20260711-005

## 任务名称
统一行情工作区 P0 权限架构修正 + P1 正确性纯函数 + AGENTS/docs 长期规则化

## 需求出处
PR #72 Review（advice.md）：接管 PR #72 并完成"修正、整合、部署、生产验证"一次性收尾。

## 修改前行为
1. `/market` URL 契约含 `debug=1`，管理员通过 `debug=1` 在 `/market` 查看原始 factor/feature/JSON，普通用户即使手工加也不可见——但调试能力与普通用户页面混在同一路由。
2. `ResearchContextPanel` 在 `debug=1` 时额外渲染 `AdminFactorDebugPanel`，且无 `LatestEventCard`/`KeyRangeCard`。
3. `EventExplanationCard`/`StructureSummaryCard` 内联提取 DTO 字段，无纯函数抽象，DTO 路径未校验。
4. `returnTo` 参数无安全校验，可被构造为外部 URL 或 `javascript:` 协议。
5. AGENTS.md §12.14 含阶段名称（"阶段二确立"/"阶段三+阶段四+最终对齐"等）、引用已删除文件（WatchlistPage.tsx/IndexPage.tsx）。
6. docs/current/04/05 含阶段名称和 `debug=1` URL 契约。
7. 无 `/admin/stock-debug` 独立管理员调试路由。
8. 无 `buildStructureSummary`/`buildUserEventExplanation` 纯函数。
9. 无 `normalizeInternalReturnTo` returnTo 安全校验。

## 修改后行为
1. `/market` URL 契约移除 `debug`；`/market?debug=1` 管理员访问时重定向到 `/admin/stock-debug/:symbol`，普通用户忽略并清除。
2. `ResearchContextPanel` 普通模式只渲染 `EventExplanationCard`/`LatestEventCard`/`StructureSummaryCard`/`KeyRangeCard`，不挂载 `StockStructuralStatePanel`。
3. 新建纯函数 `buildStructureSummary`（从 `primary[timeframe].cost_position` 等真实 DTO 路径提取结构状态摘要）和 `buildUserEventExplanation`（只消费白名单字段 + instrument mismatch 校验）。
4. 新建 `normalizeInternalReturnTo`，仅允许 `/screener`、`/market`、`/messages` 前缀，拒绝外部 URL/`javascript:`/双斜杠/非白名单前缀/超长字符串。
5. 新建 `/admin/stock-debug` 和 `/admin/stock-debug/:symbol` 路由（AdminRoute + AdminAppShell），渲染 `AdminStockDebugPage`，复用共享研究组件 + `AdminFactorDebugPanel` + `StockStructuralStatePanel`（debug=true）。
6. AGENTS.md §12.14 重写为长期规则：删除阶段名称、删除已删除文件引用、修复 debug URL 矛盾、新增管理员调试路由独立规则、纯函数规则、returnTo 安全校验规则。
7. docs/current/04 更新：移除阶段名称、移除 `debug=1` URL 契约、新增 `/admin/stock-debug` 路由文档、新增纯函数文档、新增 `normalizeInternalReturnTo` 文档。
8. docs/current/05 更新：移除阶段名称、更新测试描述（debug 测试移除、normalizeInternalReturnTo 测试新增、纯函数测试新增）。
9. docs/current/code-doc-alignment 新增 ALIGN-040（统一行情工作区代码未部署/生产未验收）。
10. docs/maps/frontend-route-map 新增 `/admin/stock-debug` 路由。
11. `EventExplanationCard` 新增 `currentInstrumentId` prop，instrument mismatch 时隐藏价格。
12. `StructureSummaryCard` 重构为使用 `buildStructureSummary` 纯函数。
13. `/stock/:symbol` 保留为兼容适配器（已复用共享 `StockResearchWorkspace`），header 视觉不统一记录为 ALIGN-040 Known Gap。
