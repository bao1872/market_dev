# CHANGE-20260709-011: tests mypy 债务清零

## 变更编号

CHANGE-20260709-011

## 任务名称

backend/tests/ mypy 债务清零 + conftest typed fixture 基础设施

## 需求出处

mypy app baseline 清零后（CHANGE-009），`backend/tests/` 仍有 300 个 mypy 错误（56 文件），不在 baseline 管控范围但鼓励逐步修复。本 PR 清零 tests mypy 债务，与 app 保持一致。

## 修改前行为

- `backend/tests/` 133 个源文件，mypy 报 300 个错误（56 文件）；
- 错误模式分布：[misc] 104（异步工厂 fixture await）、[arg-type] 63（ASGITransport / json.loads / UUID / date|None）、[union-attr] 37（None 属性访问）、[index] 26（dict|None）、[call-overload] 16、[operator] 14、[attr-defined] 12（FromClause.indexes / Coroutine / mock）；
- conftest.py 异步工厂 fixture 返回类型标注为 `Callable[..., T]`，调用方 `await factory(...)` 触发 [misc]；
- `httpx.ASGITransport(app=app)` 触发 [arg-type]（httpx 存根用 dict，Starlette 存根用 MutableMapping）；
- 大量 `json.loads(x)` / `x[0]` / `x.attr` 在 Optional 上下文未显式收窄。

## 修改后行为

- `backend/tests/` mypy 0 errors（133 source files）；
- `backend/app/` mypy 仍 0 errors（249 source files）；
- conftest.py 新增 `AsyncFactory[T]` 类型别名（`Callable[..., Coroutine[Any, Any, T]]`）和 `make_asgi_transport(app)` helper；
- 所有异步工厂 fixture（role_factory / user_factory / instrument_factory / subscription_factory / make_user_eligible / invite_code_factory）返回类型改为 `AsyncFactory[T]`；
- 所有测试文件中 `Callable[..., T]` 参数标注改为 `AsyncFactory[T]`；
- Optional 分支用显式 `assert x is not None` 收窄；
- `app/services/access_control_service.py`：`require_feature` / `require_quota` 返回类型从 `Callable[..., object]` 收紧为 `Callable[..., Coroutine[Any, Any, AccessContext]]` / `Callable[..., Coroutine[Any, Any, int | None]]`（类型-only，无运行时行为变化，内部函数本就是 async）；
- `app/strategy/runtime.py`：`StrategyRuntime.execute` 返回类型从 `StrategyResult` 收紧为 `StrategyResult | None`（诚实 typing，batch service 已在 line 843 处理 None）；
- `calculate_metrics` 新增 `assert result is not None`（无生产调用方）。

## Any/cast/type:ignore 使用说明

本 PR 使用 1 处 `cast`，有明确理由：

- `backend/tests/conftest.py::make_asgi_transport`：`cast(_asgi_app, app)` 桥接 httpx ASGITransport 存根（用 `dict[str, Any]`）与 Starlette/FastAPI `__call__` 存根（用 `MutableMapping[str, Any]`）的结构子类型不兼容。这是第三方存根缺口，运行时 FastAPI 本就是合法 ASGI3 app。cast 集中在单点 helper，不在每个测试里重复。

未使用任何 `type: ignore` 或 `Any` 掩盖测试错误。
