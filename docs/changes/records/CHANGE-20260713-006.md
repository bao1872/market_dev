## 修改前行为

1. `/market` 顶部 `MarketToolbar` 仅剩 scope 分段 + 搜索框，行业/概念筛选在 CHANGE-20260713-005 中被移除（DSA API 不支持行业/概念筛选的旧判断）。
2. `/strategy-runs/{run_id}/results` API 不支持 industry/concept 参数；strategy_result_repository 无 board filter 条件。
3. `market_stocks_service._build_board_filter_conditions` 为 market_stocks 专用，未提取共享 helper。
4. `TableViewPresetConfig`（前端 + 后端）白名单不含 industry/concept 字段。
5. 视觉系统使用旧"交易终端蓝"主题：variables.scss 主色 #2962ff、背景 #0f1117/#171a22、文本 #d1d4dc/#8b93a7；BrandLogo 使用蓝紫渐变圆形 + 白色折线；global.scss、MarketWorkspace.module.scss、AccountMenu、UserAppShell、MarketInstrumentPane 含大量硬编码颜色。
6. 无盘迹品牌视觉资产包 V1.0 的莹感绿 token 体系。

## 修改后行为

### 一、行业/概念筛选恢复（DSA 列表）

1. **共享 board_filter_helper**：新建 `backend/app/repositories/board_filter_helper.py`，导出 `build_board_filter_conditions(instrument_id_col, industry, concept)`，构造 EXISTS 子查询（MarketBoardMembership JOIN MarketBoard，type='industry'/'concept'，name 匹配）。industry+concept 同时提供时为 AND 语义（两个 EXISTS 条件 AND 连接）。
2. **strategy_result_repository 复用**：`_apply_run_item_filters` 增加 `industry`/`concept` 参数，调用共享 helper（以 `StrategyRunItem.instrument_id` 为关联列）；base 和 count_base 共用同一过滤函数；`query_run_items_with_results` 签名增加 industry/concept，透传到 `_apply_run_item_filters`。
3. **selector_query_service 透传**：`query_published_selector_results` 增加 industry/concept 参数，传递给 `query_run_items_with_results`。
4. **API 暴露**：`/strategy-runs/{run_id}/results` 增加 `industry`/`concept` Query 参数（str | None），传递给 `query_published_selector_results`。
5. **market_stocks_service 委托**：`_build_board_filter_conditions` 改为委托共享 helper（以 `Instrument.id` 为关联列）；移除未使用的 MarketBoardMembership 直接导入。
6. **items/filtered_total 一致**：`items` 和 `filtered_total` 应用完全相同的 keyword/industry/concept/metric_filters 条件；`source_total` 不受业务筛选影响（恒等于 `run.total_instruments`）；SQL 数量固定，不 N+1。
7. **preset 持久化**：`TableViewPresetConfig`（后端 Pydantic + 前端 TS 接口）增加 `industry`/`concept` 可选字段（max_length=100）；`_ALLOWED_CONFIG_KEYS` 白名单增加对应键；不新增表/migration（JSONB 列）；旧 preset 缺字段时正常兼容（Pydantic default=None）；未知板块值在目录变化后忽略（EXISTS 无匹配返回空）。
8. **前端 URL state**：`marketWorkspaceUrlState.ts` 的 `MarketWorkspaceUrlState` 增加 `industry: string | null` / `concept: string | null`；decode/encode 读写 URL `industry`/`concept` 参数。
9. **MarketToolbar 恢复行业/概念输入**：渲染"搜索、行业、概念"同一行布局；行业/概念为 `<input type="search">` + `<datalist>` 候选；`boards.available=false` 时输入禁用，placeholder 显示"板块数据暂不可用"（不删除输入，保留降级提示）。
10. **MarketWorkspacePage 集成**：调用一次 `useMarketBoards`；`resultParams` 增加 industry/concept；切 scope/搜索/排序/分页时保留 industry/concept；改变板块筛选重置 page=1；handleIndustryChange/handleConceptChange 通过 URL state 更新。
11. **StrategyDataTable 受控 props**：增加 `externalIndustry`/`onIndustryChange`/`externalConcept`/`onConceptChange` props（与 externalKeyword 同模式）；`currentConfig` 包含 industry/concept 快照；`applyPresetConfig` 恢复 industry/concept 到外部受控 state。
12. **后端测试**：`test_strategy_results_industry_concept.py` 覆盖 6 个场景：industry 匹配、concept 匹配、industry+concept AND 交集、不存在板块返回空、无板块数据返回空、无筛选返回全部；断言 `len(page.items) == page.filtered_total`。

### 二、盘迹品牌视觉 V1.0

13. **variables.scss 成为唯一视觉 token 真源**：更新为 V1.0 莹感绿体系：
    - 背景：$color-bg #0A0F14（深石墨黑）、$color-panel #111A23（暗蓝灰）、$color-panel-2 #161F29（卡片背景）、$color-panel-3 #1a2433（三级面板）
    - 边框：$color-border #263440、$color-border-soft #1d2832
    - 文本：$color-text #F2F6F8（雾白）、$color-muted #98A1B3（灰蓝）、$color-text-dim #657281
    - 品牌莹感绿：$color-brand #00F6C2（主色）、$color-brand-hover #39F5CF（高亮）、$color-brand-deep #00B28A（深绿）
    - 功能色：$color-info #3882F6（提示蓝）、$color-warning #F59E0B（预警橙）、$color-purple #8B5CF6
    - A股语义：$color-up #FF4D4F（红涨）、$color-down #22C55E（绿跌）
    - 字体：$font-sans MiSans/HarmonyOS Sans SC/PingFang SC/Noto Sans CJK SC/Microsoft YaHei
    - 圆角：$radius-base 10px、$radius-card 12px
    - 阴影：$shadow-card 0 8px 24px rgba(0,0,0,0.32)（轻阴影，禁止重阴影和大面积玻璃拟态）
    - 向后兼容别名：$color-blue→$color-brand、$color-blue-2→$color-brand-hover、$color-green→$color-down、$color-red→$color-up、$color-orange/$color-yellow→$color-warning
    - CSS 自定义属性同步导出（--bg/--panel/--brand/--up/--down 等）
14. **BrandLogo 重写**：从蓝紫渐变圆形+白色折线改为四节点折线路径 + 末端高亮共识节点（莹感绿圆环）；三个非末端节点为雾白低饱和（opacity 0.7-0.92），末端节点为品牌主色 #00F6C2 高亮圆环（r=3.2 stroke）+ 实心点（r=1.8 fill）；不变形、不旋转、不增减节点、不替换颜色。
15. **品牌资产**：从 ref/盘迹品牌视觉资产包_v1.0/01_标志系统 复制 Web 必需资产到 `frontend/src/assets/brand/`：logo_symbol_128.png（5.9KB）、logo_symbol_256.png（15.5KB）、logo_horizontal_dark.png（58.8KB）；不复制字体、视频和整包素材；ref 只作设计真源，运行时代码不依赖 ref 路径。
16. **硬编码颜色清理**：
    - `global.scss`：30 处替换（#131722→v.$color-panel、#0f1117→v.$color-bg、#2b3040→v.$color-border、#687083→v.$color-text-dim、#8b93a7→v.$color-muted、#2962ff→v.$color-brand、#4b7cff→v.$color-brand-hover 等）

    - `AccountMenu.module.scss`：#131722→v.$color-panel、#1c202b→v.$color-panel-3；新增 variables 导入
    - `UserAppShell.module.scss`：#2962ff→v.$color-brand、#1c202b→v.$color-panel-3；新增 variables 导入
    - `MarketInstrumentPane.module.scss`：#2962ff→v.$color-brand、#1c202b→v.$color-panel-3；新增 variables 导入
