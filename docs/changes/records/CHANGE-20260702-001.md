## 3. 修改后设计

- 不新增或重命名规则目录中的文件；删除原包中不属于既定结构的 `docs/generated/` 与 `docs/templates/`；

- `18-code-doc-alignment.md` 只登记已确认设计与实现差异；
- 套餐事实源改为 `plans` + `plan_service`；
- PostgreSQL 改为 Compose 容器；
