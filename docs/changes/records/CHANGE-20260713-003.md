## 修改前行为
1. `MarketToolbar` 和 `MarketStockTable` 各自调用 `useMarketBoards`，存在重复请求。
2. `MarketStockTable` 显示"最近事件"列；`market_stocks_service` 执行 `stock_state_event` 批量查询（9 条固定 SQL）组装 `latest_event`。
3. 形态状态/DSA 状态在表格中直接显示内部 code（如 `between_nodes`）。
4. `StrategyChart` 渲染 `tv-strategy-legend` 只读说明行，与 `IndicatorToolbar` 形成两套图层开关源；`DISPLAY_GROUPS`/`DisplayGroupDef` 定义存在且唯一消费者是 legend。
5. MACD 在 `watchlist` 和 `selection` 默认开启（`macd: true`），但 MACD 不是 bar 因子也不是时序特征。
6. `/market` 和 `/stock/:symbol` 右栏首次默认展开，收起状态未持久化。
7. AGENTS.md 规则 16/17/18 引用已删除组件（ResearchContextPanel/EventExplanationCard/StructureSummaryCard/KeyRangeCard/LatestEventCard/AdminFactorDebugPanel/useResearchContext/buildStructureSummary/buildUserEventExplanation）。

## 修改后行为
1. **boards 单一真源**：`MarketWorkspacePage` 唯一调用 `useMarketBoards`，将 `boardsAvailable`/`industryOptions`/`conceptOptions` 以 props 传给 `MarketToolbar` 和 `MarketStockTable`；子组件不再请求 boards。
2. **行业/概念列条件渲染**：`boardsAvailable=false` 时 `MarketToolbar` 禁用行业/概念输入并显示"板块未开放"，`MarketStockTable` 隐藏行业/概念列；`boardsAvailable=true` 时恢复显示。
3. **删除最近事件列**：`MarketStockTable` 不再显示"最近事件"列；`market_stocks_service` 删除 `stock_state_event` 批量查询、`event_map` 组装逻辑（固定 SQL 数从 9 条减为 8 条）；`MarketStockRow.latest_event_title`/`latest_event_time` 兼容保留为 `null`；事件只在 `EventStatePanel` 按需展开时通过 `useStockContext` 加载。
4. **形态状态中文映射**：新增 `mapStructureStateLabel.ts` 纯函数（`mapStructureStateLabel` + `mapDsaStateLabel`），将 code 映射为中文展示；`price_as_of`/`state_as_of` 放到表格底部数据日期提示，不逐行重复。
5. **删除 tv-strategy-legend**：`StrategyChart` 删除 `tv-strategy-legend` JSX 整行 + `isGroupActive` 函数 + `DISPLAY_GROUPS`/`DisplayGroupDef`/`DSA_TITLE_HINT` imports + 相关 CSS；`strategy-manifest.ts` 删除 `DISPLAY_GROUPS`/`DisplayGroupDef` 定义；图层开关唯一真源在 `IndicatorToolbar`。
6. **MACD 语义纠正**：`stock_state.py` 注释更新为"MACD 是 feature_snapshot_service 附加的日线辅助技术指标（标准 12/26/9），注入 primary_factors.macd_state；不是 bar 因子也不是时序特征"；`defaultChartLayerVisibility` 中 `watchlist` 和 `selection` 默认 `macd: false`；不删历史字段、不迁移、不回填。
7. **右栏默认收起**：`/market`（`panji:market-right-panel-collapsed:v1`）和 `/stock/:symbol`（`panji:event-panel:v1`）首次默认收起右栏，localStorage 持久化用户选择；收起时不挂载 `EventStatePanel`、不请求 `useStockContext`。
8. **AGENTS.md 规则更新**：规则 8/11/16/17/18 重写为反映当前实现（EventStatePanel + useStockContext）；新增规则 19（图表图层 ≠ 策略因子）/20（MACD 语义）/21（右栏按需加载）/22（boards 单一真源）/23（latest_event 兼容保留）。
