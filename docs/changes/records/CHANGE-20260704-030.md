# CHANGE-20260704-030

## 变更编号

CHANGE-20260704-030

## 任务名称

个股详情页新增 SQZMOM_LB 指标图层

## 需求出处

- 用户要求实现 TradingView Pine `SQZMOM_LB`（Squeeze Momentum Indicator by LazyBear）。
- 关键约束：后端计算逻辑必须与 Pine 代码逐行等价，不修正原脚本疑似 bug（`dev = multKC * stdev(source, length)`）。
- 目标：在个股详情页 `/stock/:symbol` 增加 SQZMOM_LB 指标开关和副图展示。

## 修改前行为

- 个股详情页技术指标区仅有 MACD 副图，无 SQZMOM_LB。
- `/api/v1/instruments/{instrument_id}/indicators` 返回 `layers` 和 `data` 中均无 `sqzmom_lb`。
- 后端无 SQZMOM_LB 算法实现。
- 前端无 SQZMOM 图层开关和副图渲染器。

## 修改后行为

- 后端新增 `app.strategy_assets.algorithms.features.sqzmom_lb.compute_sqzmom_lb`，逐行复刻 Pine 代码：
  - `basis = sma(source, length)`；
  - `dev = multKC * stdev(source, length)`（保持原脚本，不修正为 `mult * stdev`）；
  - `ma = sma(source, lengthKC)`；
  - `range = useTrueRange ? tr : high - low`；
  - `rangema = sma(range, lengthKC)`；
  - `upperKC = ma + rangema * multKC`、`lowerKC = ma - rangema * multKC`；
  - `sqzOn`/`sqzOff`/`noSqz` 判断；
  - `val = linreg(source - avg(avg(highest(high, lengthKC), lowest(low, lengthKC)), sma(close, lengthKC)), lengthKC, 0)`；
  - `bcolor`/`scolor` 按 Pine 颜色逻辑输出；
  - 默认参数 `length=20, mult=2.0, lengthKC=20, multKC=1.5, useTrueRange=true`。
- `indicator_service.compute_all_indicators` 中直接调用并注入 `sqzmom_lb` 数据与图层，不走 StrategyRuntime 注册路径（与 MACD 全局图层一致）。
- `/api/v1/instruments/{instrument_id}/indicators` 响应新增：
  - `data.sqzmom_lb`：含 `val`、`sqzOn`、`sqzOff`、`noSqz`、`bcolor`、`scolor`、`time`、`params`；
  - `layers` 中新增 `strategy_id=sqzmom_lb`、`renderer=sqzmom`、`pane=sqzmom` 图层定义。
- 前端 `strategy-manifest.ts` 新增 `sqzmom` 图层定义和 `sqzmom` DISPLAY_GROUP。
- 前端 `StrategyChart.tsx` 新增：
  - `LayerVisibility.sqzmom`；
  - 独立 `sqzmom` pane 分配（位于 MACD 上方）；
  - `renderIndicatorSqzmom` 渲染器：histogram 使用 `val` + `bcolor`，0 轴 squeeze marker 使用 `scolor`；
  - 默认关闭、API 缺失时不崩溃的 guard。
- 测试：
  - 后端 `backend/tests/test_sqzmom_lb.py` 21 个测试覆盖 Pine 等价、BB dev 使用 multKC、sqzOn/sqzOff/noSqz、linreg offset=0、nz(val[1]) 颜色逻辑、数据不足不 500；
  - 后端 `backend/tests/test_indicator_service.py` 追加 indicator_service 注入测试；
  - 前端 `frontend/scripts/contract-tests/sqzmom-layer.test.ts` 5 个 contract test 覆盖默认关闭、renderer 注册、pane 分配、API 缺失不崩溃、前端不重新计算指标。

- 前端 SQZMOM 副图高度固定 82px，与 MACD 一致；若未来同时开启多个副图，需评估垂直空间。
