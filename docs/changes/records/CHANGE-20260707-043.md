# CHANGE-20260707-043: Indicator Overlay Frontend Hardcode Cleanup

## 变更编号
CHANGE-20260707-043

## 任务名称
PR #33 — 彻底移除 StrategyChart 残留的 4 处 1d-only / 1w-1mo skip 硬编码，让 DSA/BB 全周期按开关正常显示/隐藏

## 需求出处
PR #32 汇报说 DSA 全周期可点、BB 1w/1mo 可画，但 main 当前 StrategyChart 仍有 4 处旧硬编码：
1. `handleToggleGroup` L2226: `if (groupId === 'dsa' && timeframe !== '1d') return` — 非 1d DSA toggle 不能切换
2. render loop L1661: `if (layer.layer_id === 'dsa_vwap' && timeframe !== '1d') return` — 15m/1h/1w/1mo DSA 即使有数据也不画
3. render loop L1666: `if (layer.layer_id === 'bb' && (timeframe === '1w' || timeframe === '1mo')) return` — 1w/1mo BB 直接不画
4. priceCandidates L1503: `if (layer.layer_id === 'dsa_vwap' && layers.dsa && timeframe === '1d')` — DSA 纵轴范围仅 1d 纳入

明确约束：
- 不改 DSA/BB 数学公式
- 不做事件/相似场景/板块聚合
- 不为显示而绕过 source mismatch 校验
- 不生成截图/coverage/大日志
- 不新增表/worker/全市场循环

## 修改前行为
- L2226: DSA toggle 在非 1d 周期被硬编码 disable，`shouldToggleDsa` 不存在
- L1661: DSA render 在非 1d 周期被硬编码 skip，`shouldRenderDsaLayer` 不存在
- L1666: BB render 在 1w/1mo 被硬编码 skip，`shouldRenderBbLayer` 不存在
- L1503: DSA 纵轴范围候选仅 `timeframe === '1d'` 纳入，`shouldIncludeDsaInPriceRange` 不存在
- `dsaOverlayPolicy.ts` 仅有 `shouldAllowDsaOverlay` / `shouldCheckDsaMismatch` / `DSA_TITLE_HINT`

## 修改后行为
- 新增 5 个纯函数决策到 `frontend/src/utils/dsaOverlayPolicy.ts`：
  - `shouldAllowBbOverlay(timeframe)`: BB 全周期支持（1d/15m/1h/1w/1mo）
  - `shouldRenderDsaLayer(layerId, layers, dsaSourceMismatch, timeframe)`: DSA 渲染决策（开关 / source mismatch / 周期支持三要素）
  - `shouldRenderBbLayer(layerId, layers, timeframe)`: BB 渲染决策（开关 / 周期支持两要素）
  - `shouldToggleDsa(groupId, isCaptureMode, captureLayers)`: DSA toggle 决策（capture 锁定，非 capture 全周期可切换）
  - `shouldIncludeDsaInPriceRange(layerId, layers, timeframe)`: DSA 纵轴范围候选决策（全周期参与）
- `StrategyChart.tsx` 4 处硬编码全部替换为决策函数调用：
  - L2226: `if (!shouldToggleDsa(groupId, isCaptureMode, FEISHU_CAPTURE_LAYERS)) return`
  - L1660-1663: `if (layer.layer_id === 'dsa_vwap' && !shouldRenderDsaLayer(...)) return`
  - L1663: `if (layer.layer_id === 'bb' && !shouldRenderBbLayer(...)) return`
  - L1504: `if (shouldIncludeDsaInPriceRange(layer.layer_id, layers, timeframe))`
- 保留 source mismatch 保护：`shouldRenderDsaLayer` 在 `dsaSourceMismatch=true` 时全周期返回 false
- 保留 capture 锁定：`shouldToggleDsa` 在 capture 模式锁定 DSA 不可关闭
- DSA toggle 全周期可切换：`shouldToggleDsa('dsa', false, FEISHU_CAPTURE_LAYERS)` 全周期返回 true
- DSA 渲染全周期支持：`shouldRenderDsaLayer('dsa_vwap', {dsa:true}, false, tf)` 全周期返回 true
- BB 渲染 1w/1mo 不再 skip：`shouldRenderBbLayer('bb', {bb:true}, '1w'/'1mo')` 返回 true
- DSA 全周期参与 y-axis：`shouldIncludeDsaInPriceRange('dsa_vwap', {dsa:true}, tf)` 全周期返回 true
