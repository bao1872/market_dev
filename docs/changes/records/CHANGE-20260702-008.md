## 2. 修改前理解

- `indicator_contract.py` 中 `NODE_CLUSTER_LOW_BARS=3600`，应为 `250*16=4000`。
- `INDICATOR_BARS["15m"]=3600` 硬编码，与 `NODE_CLUSTER_LOW_BARS` 数值耦合，注释写"与 Node Cluster 低周期一致"。
- 实际 15m 截断由 `NODE_CLUSTER_LOW_BARS` 在 `prepare_node_cluster_bars` 中完成；`INDICATOR_BARS["15m"]` 仅作为 `calculation_window` 元信息返回前端。
- `monitor_batch_service.py` 引用 `IC.NODE_CLUSTER_LOW_BARS`（不是 `INDICATOR_BARS["15m"]`）。
- `unified_volume_profile.py` 第 272 行 docstring 陈旧，写 `tail(1200)` 但实际用 `NODE_CLUSTER_LOW_BARS`。
- `StrategyVersion` 不记录 3600（manifest 中无 15m bar count 参数）。
- 前端只传 `page_size=250`，不感知 3600。

## 3. 修改目标

- `NODE_CLUSTER_LOW_BARS` 从 3600 改为 `DAILY_HISTORY_BARS * NODE_CLUSTER_15M_BARS_PER_DAY = 4000`
- 新增 `NODE_CLUSTER_15M_BARS_PER_DAY = 16`
- `INDICATOR_BARS["15m"]` 改为引用 `NODE_CLUSTER_LOW_BARS`（不再硬编码 3600）
- 修正陈旧 docstring
- TDD 测试验证契约
